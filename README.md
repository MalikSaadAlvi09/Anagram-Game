# Anagram-Game
This repo features a dual-tiered Anagram Game in Python, designed for JupyterLab. It includes a Beginner version focusing on core string logic and a "shuffle-and-check" MVP, alongside an Advanced version with score tracking, hint systems, and dynamic difficulty. Perfect for seeing how code evolves from simple scripts to robust apps.

## 🚀 Key Features

### 🟢 Beginner Version
*   **Simple Logic:** Clean, readable code for newcomers.
*   **Core Mechanics:** Randomized shuffling and direct validation.
*   **Environment:** Runs instantly in JupyterLab (`/Anagram-Game`).

### 🔴 Advanced Version
*   **Score Tracking:** Save your high scores across sessions.
*   **Hint System:** Stuck? Reveal letters to help you solve it.
*   **Dynamic Difficulty:** Words get tougher as you win.
*   **Input Sanitization:** Robust handling of typos and casing.
